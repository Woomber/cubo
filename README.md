# Cubo

Aplicación C#.NET que dibuja un cubo dados parámetros de escala, posición, rotación y proyección.

**Nota:** Requiere la librería [Matemáticas (matrices-fracciones)](https://github.com/Woomber/matrices-fracciones-lib).
